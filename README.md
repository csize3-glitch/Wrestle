# WrestleWell

Starter monorepo scaffold for a wrestling website and mobile app.

## Quick start

1. Copy `.env.example` to `.env`
2. Add your Firebase project values
3. Put your workbook in `data/wrestling_youtube_drills_all_styles_expanded.xlsx`
4. Run `pnpm install`
5. Run `pnpm dev:web` or `pnpm dev:mobile`
6. Run `pnpm import:library` to import drills into Firestore
